# JetPack Xidmət Yol Xəritəsi

JetPack Xidmətinin vizyonu “**İstifadəçilər və Xidmətlər arasında Birgəyaşayış**”ı həyata keçirməkdir.

**Faza 1** hər mərhələdə, Mərhələ-1-dən Mərhələ-5-ə qədər əsas xüsusiyyətləri və tətbiq tamamlanma vaxtlarını təsvir edir. Bu fazada, JetPack sistemi və onun xidmətləri arasında problemsiz inteqrasiya üçün bütün zəruri funksionallıqlar inkişaf etdiriləcək, əsas AI-dən idarə olunan xidmətlərin tədricən istifadəyə verilməsinə diqqət yetiriləcək. Hər mərhələ **“AI-İlk Xidmət Təşəbbüsü”** missiyası ilə idarə olunur.

**Faza 2** JetPack Xidməti kifayət qədər dəyər göstərdikdən sonra başlayacaq. Bu faza JetPack-in token əsaslı reklam sistemini və metaverse və e-ticarət kimi üçüncü tərəf platformaları ilə inteqrasiyaları təqdim edəcək. Ətraflı cədvəl daha sonra elan ediləcək.


## Faza 1

### **Mərhələ-1: 'Qagarin' Mərhələsi**
**Başlama Tarixi**: 1 Yanvar 2025

- Xidmət idarəetmə sisteminin inkişafı
  - Xidmət qeydiyyatı və silinmə funksiyaları
  - Token paylama sisteminin tətbiqi
- Veb sayt və əsas REST API inkişafı

**Mərhələ Adlandırılıb**: Yuri Qagarin, kosmosa çıxan ilk insan

---

### **Mərhələ-2: 'Şepard' Mərhələsi**
**Gözlənilən Vaxt Çərçivəsi**: 2025-ci ilin 3-cü rübü

- **haru** gündəlik tətbiqinin istifadəyə verilməsi
- **OG Degen Bunny** token düşmə funksionallığı
- İstifadəçilər arasında JetPack token transfer funksiyası
- Xarici cüzdan dəstəyi inteqrasiyası

**Mərhələ Adlandırılıb**: Alan Şepard, kosmosa çıxan ilk amerikalı

---

### **Mərhələ-3: 'Tereşkova' Mərhələsi**
**Gözlənilən Vaxt Çərçivəsi**: 2026-cı ilin 1-ci rübü

- **Prompt Xidməti**nin istifadəyə verilməsi
- **JetPack Token Exchange (CEX)** inkişafı
- **NFT Drop Xidməti**nin inkişafı
- Çoxdilli SDK-nın genişləndirilməsi

**Mərhələ Adlandırılıb**: Valentina Tereşkova, kosmosa çıxan ilk qadın

---

### **Mərhələ-4: 'Armstrong' Mərhələsi**
**Gözlənilən Vaxt Çərçivəsi**: 2026-cı ilin 3-cü rübü

- NFT ilə əlaqəli xidmətlərin istifadəyə verilməsi
- **DAO əsaslı qaydalar üzrə səsvermə sistemi**nin inkişafı
  - Mərkəzsiz qərar qəbul etmə sistemlərinin tətbiqi

**Mərhələ Adlandırılıb**: Neil Armstrong, aya ayaq basan ilk insan

---

### **Mərhələ-5: 'Leonov' Mərhələsi**
**Gözlənilən Vaxt Çərçivəsi**: 2027-ci ilin 1-ci rübü

- **DeFi ilə əlaqəli xidmətlərin** istifadəyə verilməsi
- Xarici konfranslar vasitəsilə xidmət təminatçılarının cəlb edilməsi
  - Hədəf auditoriya: universitet tələbələri, tək inkişafçılar və s.
- Monitorinq və fırıldaqçılıq aşkarlama sisteminin (FDS) təkmilləşdirilməsi
  - Sistem sabitliyinin və təhlükəsizliyinin gücləndirilməsi

**Mərhələ Adlandırılıb**: Aleksey Leonov, kosmosda ilk dəfə gəzinti edən insan
