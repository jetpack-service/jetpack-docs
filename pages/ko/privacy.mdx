# JetPack 서비스 개인정보처리방침

JetPack 서비스는 사용자의 개인정보를 보호하고, 이를 투명하고 안전하게 처리하기 위해 최선을 다하고 있습니다. 본 개인정보처리방침은 JetPack 서비스를 이용하는 동안 제공되는 개인정보가 어떻게 수집, 사용, 저장 및 보호되는지를 설명합니다.

## 1. 개인정보의 수집 항목 및 방법

### 1.1 수집 항목
JetPack 서비스는 다음의 정보를 수집합니다:

- 이메일 주소 (Google OAuth 인증 시 제공, 수정 불가)
- 디스플레이 이름 (Google OAuth 인증 시 제공, 수정 가능)
- 지갑 정보 (카르다노 블록체인의 Public Key)

### 1.2 수집 방법
- Google OAuth를 통한 인증 시 이메일 주소와 디스플레이 이름을 자동 수집합니다.
- JetPack 서비스를 사용할 때 사용자 계정 생성 시 지갑 정보(Public Key)를 생성 및 저장합니다.

## 2. 개인정보의 이용 목적

JetPack 서비스는 수집한 개인정보를 다음과 같은 목적으로 사용합니다:

1. 사용자 인증 및 JetPack 서비스 계정 관리
2. JetPack 토큰의 발행, 거래 및 사용 관리
3. 사용자 문의 및 지원 제공
4. 디스플레이 이름을 기반으로 개인화된 사용자 경험 제공
5. 법적 규제 및 의무 준수를 위한 정보 활용

## 3. 개인정보의 보관 및 삭제

### 3.1 보관 기간
- 사용자가 회원 탈퇴를 요청하지 않는 한, 개인정보는 계정이 활성화된 동안 계속 보관됩니다.

### 3.2 지갑 정보 관리 및 주의사항
1. **지갑 생성 및 Private Key 관리**:
   - JetPack 서비스의 지갑은 회원 가입 시 1회에 한해 생성됩니다.
   - **Private Key는 사용자의 이메일로 제공**되며, JetPack 서비스는 Private Key를 저장하지 않습니다.
   - 사용자는 Private Key를 안전하게 보관해야 하며, 분실 시 복구할 수 없습니다.

2. **회원 탈퇴 및 지갑 재생성 주의사항**:
   - 회원 탈퇴 후 재가입 시 새로운 지갑이 생성되며, 이전 지갑에 남아 있던 토큰은 사용자가 복구할 수 없습니다.
   - **토큰 손실 위험**이 있으므로, 탈퇴 전 반드시 토큰을 처리하시기 바랍니다.

3. **향후 추가 지갑 지원 계획**:
   - JetPack 서비스는 향후 카르다노의 Public Key와 Private Key를 이용하여 추가 지갑을 설정할 수 있는 기능을 제공할 예정입니다.
   - 현재는 해당 기능을 지원하지 않으며, 생성된 지갑만 사용할 수 있습니다.

### 3.3 삭제 가능 정보
- 사용자는 JetPack 서비스의 **회원 탈퇴 기능**을 통해 계정을 삭제할 수 있습니다.
- 탈퇴 시 이메일과 디스플레이 이름, 지갑 정보를 매칭하는 데이터가 삭제됩니다.
- 삭제된 이후 지갑 정보는 특정 개인과 매칭되지 않으며 익명화된 상태로 남게 됩니다.

### 3.4 삭제 불가능 정보
1. **지갑 정보 및 거래내역**:
   - JetPack 서비스의 지갑 정보는 카르다노 블록체인의 Public Key로 구성되며, 생성된 지갑 정보는 삭제되지 않습니다.
   - 블록체인 상의 지갑 정보와 거래 내역은 변경하거나 삭제할 수 없으며, **영구 보관**됩니다.

2. **블록체인 상의 정보**:
   - 블록체인에 기록된 거래 내역은 데이터의 무결성을 보장하며, JetPack 서비스 생태계의 투명성을 유지하기 위해 영구히 보관됩니다.

## 4. 개인정보의 공유 및 제공

JetPack 서비스는 원칙적으로 사용자의 개인정보를 제3자와 공유하거나 제공하지 않습니다. 다만, 다음의 경우 예외적으로 제공할 수 있습니다:

- 사용자의 사전 동의를 받은 경우
- 법령에 의거하거나 법적 의무 준수를 위해 필요한 경우

## 5. 개인정보 보호를 위한 노력

JetPack 서비스는 사용자의 개인정보를 안전하게 보호하기 위해 다음과 같은 조치를 취합니다:

1. **기술적 조치**:
   - 데이터 암호화 및 전송 시 보안 프로토콜(SSL/TLS) 사용
   - 외부 공격 방지를 위한 최신 보안 시스템 적용

2. **관리적 조치**:
   - 개인정보 접근 권한 최소화
   - 정기적인 내부 보안 점검

3. **물리적 조치**:
   - 개인정보가 저장된 시스템 접근 통제
   - 안전한 데이터 보관 환경 유지

## 6. 사용자의 권리와 선택

사용자는 다음과 같은 권리를 행사할 수 있습니다:

1. **개인정보 열람 및 수정**:
   - JetPack 서비스 계정 설정에서 디스플레이 이름을 수정할 수 있습니다.
   - 이메일 주소는 OAuth 인증을 통해 수집되며 수정할 수 없습니다.

2. **회원 탈퇴 요청**:
   - JetPack 서비스의 **회원 탈퇴 기능**을 통해 계정을 삭제할 수 있습니다.
   - 탈퇴 시 기존 지갑 정보와 이메일 매칭 데이터는 삭제되며, 새로운 가입 시 다른 지갑이 생성됩니다.
   - **토큰 손실 위험**이 있으므로, 탈퇴 전에 토큰을 반드시 처리해야 합니다.

3. **OAuth 인증 철회**:
   - OAuth 인증을 철회할 경우 JetPack 서비스 이용이 제한될 수 있지만, 기존 데이터는 삭제되지 않고 유지됩니다.

- 이메일: support@jetpack.ws
