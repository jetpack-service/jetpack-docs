## **Apa manfaat token selain hadiah?**

Token JetPack akan digunakan dalam layanan yang kami kembangkan. Layanan ini akan memberi penghargaan kepada pengguna dengan token JetPack berdasarkan aktivitas mereka dan memungkinkan akses ke fitur premium dengan harga diskon dengan menggabungkan token JetPack dan uang tunai. Selain itu, semakin banyak token yang dihabiskan pengguna, semakin mudah untuk naik level dalam layanan. Seiring dengan peningkatan level, jumlah token yang dialokasikan juga bertambah, memungkinkan layanan aplikasi untuk memberikan lebih banyak token JetPack kepada pengguna. Distribusi token untuk setiap level akan dapat disesuaikan melalui Ancient Degen Bunnies pada fase tata kelola akhir.

## **Mengapa lebih menguntungkan untuk menyimpan atau menggunakan token daripada menjualnya segera?**

Kami berencana untuk membuat bursa token JetPack khusus, tetapi pencatatan di bursa besar seperti Binance tidak mungkin. Token dapat dijual melalui bursa yang kami siapkan, di mana saya akan membeli sebagian besar untuk memastikan likuiditas. Token tidak akan dibakar, dan dana pembelian akan berasal dari 20% pendapatan iklan layanan, yang akan diungkapkan secara transparan. Selain itu, untuk terdaftar di bursa besar seperti Binance, likuiditas harus terjamin, dan pada fase awal, saya kemungkinan akan menjadi pembeli tunggal token. Sampai investor besar yang percaya pada token JetPack muncul, saya akan memimpin dalam menjaga likuiditas. Mengingat potensi apresiasi nilai token, menyimpannya bisa menguntungkan.

Staking tidak dipertimbangkan, tetapi **token JetPack akan didistribusikan sebagai hadiah untuk staking NFT**. **NFT OG Degen Bunnies** akan menerima hadiah tertinggi dan dukungan juga akan diperluas ke NFT dari proyek rug pull untuk mempromosikan ekspansi komunitas.

## **Dapatkah token digunakan dalam layanan tertentu? Misalnya, dapatkah saya membayar TapTools Pro dengan JetPack?**

Awalnya, fokus akan pada layanan aplikasi yang dapat memanfaatkan token JetPack. Penggunaan dalam layanan lain belum dipertimbangkan, tetapi jika ada permintaan dari platform seperti TapTools Pro, Binance, atau Upbit, integrasi dapat ditinjau. Namun, jika kami berhasil membuktikan nilai kami, integrasi alami dengan ekosistem Cardano diharapkan. Untuk mencapai ini, dukungan dari berbagai komunitas dan pengguna akan sangat penting.

## **Apa perbedaan antara Jet dan Pack?**

1 Jet = 1.000.000 Pack. Ini mirip dengan konsep 1 ADA = 1.000.000 Lovelace. Mereka bukan dua token terpisah melainkan unit yang berbeda dari satu token.
