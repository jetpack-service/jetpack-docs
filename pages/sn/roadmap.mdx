# JetPack Service Roadmap

Chiratidzo cheJetPack Service ndechekuzadzisa “**Kugarisana pakati peVashandisi neMasevhisi**.”

**Phase 1** inotsanangura zvinhu zvakakosha uye nguva dzekupedzwa kweapp pamatanho ese, kubva paStage-1 kusvika paStage-5. Muchikamu chino, mabasa ese anodiwa ekubatanidza pasina musono pakati peJetPack system nemasevhisi ayo achagadzirwa, nekutarisa pakutanga masevhisi makuru anofambiswa neAI zvishoma nezvishoma. Chikamu chimwe nechimwe chinotungamirwa nemishoni ye **“AI-First Service Initiative”**.

**Phase 2** ichatanga kana JetPack Service yaratidza kukosha kwakakwana. Chikamu chino chichaunza JetPack’s token-based advertising system uye kubatanidzwa nemapuratifomu echitatu akadai semetaverse uye e-commerce. Chirongwa chakadzama chichaziviswa gare gare.


## Phase 1

### **Stage-1: 'Gagarin' Stage**
**Start Date**: Ndira 1, 2025

- Kuvandudzwa kwehurongwa hwekugadzirisa sevhisi
  - Mabasa ekunyoresa uye kubvisa sevhisi
  - Kuiswa kwehurongwa hwekugovera token
- Kuvandudzwa kweWebsite uye basic REST API

**Stage Named After**: Yuri Gagarin, munhu wekutanga kuenda muchadenga

---

### **Stage-2: 'Shepard' Stage**
**Expected Timeline**: Q3 2025

- Kutangwa kweapp yeharu diary
- **OG Degen Bunny** token drop functionality
- JetPack token transfer feature pakati pevashandisi
- Kubatanidzwa kwekutsigira external wallet

**Stage Named After**: Alan Shepard, muAmerican wekutanga kuenda muchadenga

---

### **Stage-3: 'Tereshkova' Stage**
**Expected Timeline**: Q1 2026

- Kutangwa kwe **Prompt Service**
- Kuvandudzwa kwe **JetPack Token Exchange (CEX)**
- Kuvandudzwa kwe **NFT Drop Service**
- Kuwedzerwa kwe multilingual SDK

**Stage Named After**: Valentina Tereshkova, mukadzi wekutanga kuenda muchadenga

---

### **Stage-4: 'Armstrong' Stage**
**Expected Timeline**: Q3 2026

- Kutangwa kwemasevhisi ane chekuita neNFT
- Kuvandudzwa kwe **DAO-based rules voting system**
  - Kuiswa kwemaitiro ekuita sarudzo pasina kutungamirwa

**Stage Named After**: Neil Armstrong, munhu wekutanga kuenda pamwedzi

---

### **Stage-5: 'Leonov' Stage**
**Expected Timeline**: Q1 2027

- Kutangwa kwemasevhisi ane chekuita neDeFi
- Kukwezva vanopa masevhisi kuburikidza nemisangano yekunze
  - Vateereri vakanangwa: vadzidzi veyunivhesiti, vanogadzira vega, nezvimwe.
- Kuvandudzwa kwehurongwa hwekuongorora uye kuona kubiridzira (FDS)
  - Kusimbisa kugadzikana uye kuchengetedzeka kwehurongwa

**Stage Named After**: Alexei Leonov, munhu wekutanga kuita spacewalk
