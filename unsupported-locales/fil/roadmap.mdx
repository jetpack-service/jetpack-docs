# Roadmap ng Serbisyo ng JetPack

Ang bisyon ng JetPack Service ay maisakatuparan ang “**Pagkakaisa ng mga Gumagamit at Serbisyo**.”

**Phase 1** ay naglalarawan ng mga pangunahing tampok at mga timeline ng pagkumpleto ng app sa bawat yugto, mula Stage-1 hanggang Stage-5. Sa yugtong ito, ang lahat ng kinakailangang pag-andar para sa walang putol na integrasyon sa pagitan ng sistema ng JetPack at ng mga serbisyo nito ay bubuuin, na may pokus sa paglulunsad ng mga pangunahing serbisyong pinapagana ng AI nang paunti-unti. Ang bawat yugto ay ginagabayan ng misyon ng **“AI-First Service Initiative”**.

**Phase 2** ay magsisimula kapag ang JetPack Service ay nagpakita ng sapat na halaga. Sa yugtong ito, ipakikilala ang token-based advertising system ng JetPack at mga integrasyon sa mga third-party na platform tulad ng metaverse at e-commerce. Ang detalyadong iskedyul ay iaanunsyo sa ibang pagkakataon.


## Phase 1

### **Stage-1: 'Gagarin' Stage**
**Petsa ng Pagsisimula**: Enero 1, 2025

- Pagbuo ng sistema ng pamamahala ng serbisyo
  - Mga pag-andar ng pagpaparehistro at pagtanggal ng serbisyo
  - Pagpapatupad ng sistema ng pamamahagi ng token
- Pagbuo ng website at pangunahing REST API

**Pangalan ng Yugto Mula Kay**: Yuri Gagarin, ang unang tao sa kalawakan

---

### **Stage-2: 'Shepard' Stage**
**Inaasahang Timeline**: Q3 2025

- Paglulunsad ng app na **haru** diary
- Pag-andar ng token drop ng **OG Degen Bunny**
- Tampok ng paglipat ng JetPack token sa pagitan ng mga gumagamit
- Pagsasama ng suporta sa panlabas na wallet

**Pangalan ng Yugto Mula Kay**: Alan Shepard, ang unang Amerikanong nasa kalawakan

---

### **Stage-3: 'Tereshkova' Stage**
**Inaasahang Timeline**: Q1 2026

- Paglulunsad ng **Prompt Service**
- Pagbuo ng **JetPack Token Exchange (CEX)**
- Pagbuo ng **NFT Drop Service**
- Pagpapalawak ng multilingual SDK

**Pangalan ng Yugto Mula Kay**: Valentina Tereshkova, ang unang babaeng nasa kalawakan

---

### **Stage-4: 'Armstrong' Stage**
**Inaasahang Timeline**: Q3 2026

- Paglulunsad ng mga serbisyong may kaugnayan sa NFT
- Pagbuo ng isang **DAO-based rules voting system**
  - Pagpapatupad ng mga desentralisadong sistema ng paggawa ng desisyon

**Pangalan ng Yugto Mula Kay**: Neil Armstrong, ang unang tao na nakatapak sa buwan

---

### **Stage-5: 'Leonov' Stage**
**Inaasahang Timeline**: Q1 2027

- Paglulunsad ng mga serbisyong may kaugnayan sa DeFi
- Pag-akit ng mga tagapagbigay ng serbisyo sa pamamagitan ng mga panlabas na kumperensya
  - Target na audience: mga estudyante sa unibersidad, solo na mga developer, atbp.
- Pagpapahusay ng sistema ng pagmamanman at pagtuklas ng pandaraya (FDS)
  - Pagpapalakas ng katatagan at seguridad ng sistema

**Pangalan ng Yugto Mula Kay**: Alexei Leonov, ang unang tao na nagsagawa ng spacewalk
