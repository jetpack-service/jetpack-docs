# Sarrera

Gaur egungo paisaia digitalean, erabiltzaileek interneteko zerbitzu eta aplikazio ugarirekin parte hartzen dute egunero. Hala ere, haien parte-hartze aktiboaren arren, erabiltzaileek sarritan ez dute ia ordainik jasotzen haien ekarpen eta interakzioengatik. Aldi berean, zerbitzu-hornitzaileek, bereziki garatzaile independenteek eta startup-ek, erronka handiak dituzte puntuen sistema propioak ezartzeko eta kudeatzeko, garapen-kostu handiak eta konplexutasuna direla eta. Dauden puntuen sistema partekatuek sarrera-oztopo zorrotzak dituzte, likidazio konplikatuak, eta diru-sarreren galera potentzialak, zerbitzu batetik irabazitako puntuak beste batean erabiltzen direnean. Gainera, token sistema tradizionalek joera dute bidegabeak izateko, token kopuru handiak esleitzen dizkiete hasierako inbertitzaile eta garatzaileei, erabiltzaile arruntak hasieratik desabantailan utziz.

**JetPack Token System** irtenbide gisa agertzen da arazo hauei aurre egiteko, erabiltzaileei eta zerbitzu-hornitzaileei onura ekartzen dien ekosistema bidezko, garden eta eraginkor bat aurkeztuz.

## Arazoen Adierazpena

- **Erabiltzaileen Sarien Gabezia**: Erabiltzaileek ez dute ordain bidezkorik jasotzen interneteko zerbitzu eta aplikazioetan duten parte-hartze eta jarduerengatik, eta horrek haien motibazioa mugatzen du parte hartzeko.
- **Zerbitzu-hornitzaileentzako Erronkak**: Etxeko puntuen sistema bat garatzea eta mantentzea baliabide intentsiboa da, denbora eta inbertsio ekonomiko handia eskatzen du, eta finantza-erantzukizun bihur daiteke.
- **Dauden Puntuen Sistemen Mugak**: Puntuen sistema partekatuek sarritan sarrera-eskakizun murriztaileak dituzte, eta erabiltzaileek zerbitzu batetik irabazitako puntuak beste batean gastatzen dituztenean, jatorrizko zerbitzuaren irabaziak murriztu eta likidazio prozesu konplikatuak sor ditzakete.
- **Token Sistema Tradizionalen Bidegabekeria**: Token sistema askok desproportzionalki mesede egiten diete hasierako inbertitzaile eta garatzaileei, token kopuru handiak aurrez esleituz, lehiaketa bidezkoa eta aukera berdintasuna eragotziz erabiltzaile arruntentzat.

## Irtenbidearen Ikuspegi Orokorra

**JetPack Token System**-ek sarien eta puntuen sistema digitaletarako ikuspegi berritzailea eskaintzen du:

- **Token Banaketa Eredu Bidezko eta Gardena Ezartzea**: Parte-hartzaile guztiek aukera berdinetan hasten dira, ez baitago tokenen aurrez esleipenik inongo alderdirentzat, garatzaile eta hasierako inbertitzaileak barne.
- **Zerbitzu Anitzetan Erabil daitekeen Token Polibalentea Eskaintzea**: Erabiltzaileek JetPack Tokenak irabazi eta erabil ditzakete zerbitzu ugaritan, tokenen erabilgarritasuna eta balioa handituz.
- **Transakzio Abiadura Handia eta Segurtasuna Bermatzea**: Datu-base banatuak eta blockchain teknologia konbinatuz, JetPack-ek transakzio prozesamendu abiadura handiak (TPS) lortzen ditu, datuen osotasuna eta segurtasuna mantenduz.
- **Hazkundea Sustatzeko Zerbitzu Maila Sistema Bat Aurkeztea**: Zerbitzuek bost egunero tokenak jasotzen dituzte (epoka bat), eta haien maila erabiltzaileen token erabileraren arabera handitzen da, horrek token gehiago jasotzea eta erabiltzaileen parte-hartzea sustatzea ahalbidetuz.

## Balio Proposamen Berezia

- **Parte-hartzaile Guztientzako Aukera Bidezko eta Berdina**: JetPack-ek token sistema tradizionaletan ikusten diren abantaila bidegabeak ezabatzen ditu, tokenak aurrez inongo entitateri esleitu gabe.
- **Zerbitzu-hornitzaileentzako Integrazio Sinplifikatua**: Garatzaileek erraz integra dezakete JetPack Token System beren zerbitzuetan, emandako SDK eta APIak erabiliz, denbora eta baliabideak aurreztuz.
- **Erabiltzaileen Parte-hartze eta Sari Hobetua**: Erabiltzaileek zuzenean jasotzen dute ordaina beren jarduerengatik, haien motibazioa handituz zerbitzuekin parte hartzeko eta interakzionatzeko.
- **Transakzio Eraginkor eta Seguruak**: Datu-base banatuen eta blockchainaren konbinazioak transakzioak azkar eta segurtasunez prozesatzen direla bermatzen du, datuen osotasuna arriskuan jarri gabe.
- **Zerbitzuentzako Hazkunde Sustatzaileak**: Maila sistemak erabiltzaileen parte-hartzea eta token erabilera sustatzen duten zerbitzuak saritzen ditu, ekosistema osasuntsu eta lehiakor bat sustatuz.

## Nola Funtzionatzen Du

- **Token Banaketa eta Maila Sistema**: Bost egunero, zerbitzuek JetPack Tokenen banaketa jasotzen dute. Zerbitzuek 0tik 99ra igo dezakete haien maila, erabiltzaileek erabilitako token kopuruaren arabera, hurrengo banaketetan token gehigarriak jasotzea ahalbidetuz.
- **Erabiltzaileen Sari Mekanismoa**: Erabiltzaileek JetPack Tokenak irabazten dituzte zerbitzuetan dituzten jardueren bidez. Token hauek zerbitzu ekosisteman gastatu edo ADA trukatu ditzakete, haien interakzio digitalengatik balio ukigarria eskainiz.
- **Tokenen Erosketa eta Birbanaketa**: Zerbitzu batek esleitutako tokenak erabili ondoren, ADA erabiliz token gehigarriak erosi ditzake erabiltzaileei saritzen jarraitzeko, sari ziklo jarraia bermatuz.
- **Transakzio Prozesamendu Eraginkorra eta Segurtasun Neurriak**: Transakzio datuak datu-base banatu batean gordetzen dira eta hash kate egitura erabiliz babesten dira manipulazioa saihesteko. Gainera, bost minuturo, hash balio bat Cardano blockchainan gordetzen da datuen osotasuna frogatzeko.

## Helburu Bezeroak

- **Zerbitzu-hornitzaileak**: Garatzaile independenteak eta startup zerbitzu operadoreak, erraz integra daitekeen, eraginkorra eta segurua den puntu sistema baten bila.
- **Zerbitzu Erabiltzaileak**: Haien jarduerengatik ordain bidezkoa jaso nahi duten eta irabazitako puntuak zerbitzu ugaritan erabiltzeko malgutasuna izan nahi duten erabiltzaile orokorrak.
- **Ekosistema Ekarleak**: Nodo operadoreak, NFT mailegu-emaileak eta JetPack ekosistemaren hazkundea eta mantentzea sustatzen duten beste parte-hartzaile batzuk.

## Kanalak eta Integrazioa

- **SDK eta APIak**: Plataforma anitzetarako eskuragarri, Java, Unity eta JavaScript barne, zerbitzu eta aplikazio ugaritan integrazio erraza ahalbidetuz.

- **Zerbitzu eta Aplikazio Lankidetzak**: JetPack Tokenak erabiltzen dituzten zerbitzuekin lankidetzak, ekosistema zabaltzeko eta tokenen erabilgarritasuna handitzeko.

## Diru-sarrera Iturriak

- **Tokenen Salmenta ADAren truke**: Zerbitzuek esleitutako tokenak agortu ondoren ADA erabiliz token gehigarriak erosten dituztenean sortutako diru-sarrerak.
- **Transakzio Tasak**: Ekosisteman token transakzioetatik bildutako tasa txikiak.
- **Zerbitzu Premiumak**: Estatistika analisi profesionala bezalako zerbitzu aurreratuak eskaintzea, kuota baten truke.

## Kostu Egitura

- **Zerbitzari Azpiegitura Mantentzea**: Datu-base banatua operatzeko eta sarearen osotasuna mantentzeko lotutako kostuak.
- **Garapen eta Operazio Gastuak**: Plataforma, SDK eta APIen garapen eta mantentze etengabea.
- **Marketina eta Komunitate Kudeaketa**: Erabiltzaile eta zerbitzu-hornitzaileak eskuratzearekin eta komunitatearekin harremanetan jartzearekin lotutako gastuak.
- **Lege eta Arau Betetze Kostuak**: Aholkularitza juridikorako tasak eta arau-eskakizunak betetzea bermatzeko.

## Funtsezko Metrikak

- **Zerbitzu Parte-hartzea**: JetPack Token System-a hartzen duten zerbitzuen kopurua eta haien mailak.
- **Token Zirkulazioa eta Erabilera**: Erabiltzaileei banatutako token kopuru osoa eta zerbitzuetan token erabileraren bolumena.
- **Token Erosketa Bolumena**: Zerbitzuek ADA erabiliz erositako token kopuru osoa.
- **Erabiltzaileen Parte-hartze Metrikak**: Erabiltzaile aktiboen kopuruak, token transakzio bolumenak eta erabiltzaileen parte-hartzearen beste adierazle batzuk.
